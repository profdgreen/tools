# tools

A collection of small CLI utilities.

## Contents

| Tool | Description |
|------|-------------|
| [lt](lt/) | A smarter `tree` — trims long directory listings to keep output readable |

## Install

Each tool is self-contained in its own subdirectory. Symlink or copy the ones you want onto your `PATH`:

```bash
git clone git@git.acadian-asset.com:dgreen/tools.git ~/tools
ln -s ~/tools/lt/lt ~/.local/bin/lt
```
